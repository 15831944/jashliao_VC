
功能：

通过安装鼠标，键盘HOOK，达到实现劫获输入的密码的功能。

以移动鼠标到*号处显示*号真实密码，

主要技术：SetWindowsHookEx,EnumChildWindows,SendMessage

应用平台：98/2000/xp/2003测试通过

开发坏境：VC++6.0

使用方法：

运行本程序后，在键入任何密码时，就能自动劫获密码，

如果不能，用鼠标指向密\n码框就能自动取得密码。

并在编辑框中显示出来。
