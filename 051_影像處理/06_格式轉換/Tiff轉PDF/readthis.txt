To build this project easy, extract and build in the directory format it is in.

You must make sure that the tifflib directory is configured in your include directory list, 
in the msvc 6.0 studio go to tools->options->directories->include and point it to the correct path

same for the lib file.

Build instuructions are as follows

1) open up tifftopdf.dsw
2) build tifflib
3) build tifftopdf

4) enjoy.
