為何連上DCS-900W後，在「Active X」及「Java」內看不到影像?

By Salinna

 

本文以XP系統為例

 

1、在IE瀏覽器上方工具列點選「工具」à「網際網路選項」

 

2、點選上方「安全性」à「網際網路」à「自訂層級」

 

3、將有關「Active X」之選項皆選擇「啟用」

 

4、有關「Java」的部分也皆選「啟用」

 

5、到網址http://www.java.com/點選「Java 軟體下載」

 

6、下載完成後您點選「Java」時它即會自動安裝，

安裝完成後即可在此選項內看到即時攝影的影像囉~
