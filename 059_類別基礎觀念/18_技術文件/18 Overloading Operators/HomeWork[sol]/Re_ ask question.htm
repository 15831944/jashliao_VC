Based on your design, you can use following instructions to do operation + for this kind of objects.
 
        matrix1 A(1,2),B(1,2);
        matrix1 *C=A+&B;            <===> matrix *c= A.operator+(&B)
 
It's ok!   ( Why & ??: Since you define the operator+ method which needs a pointer type paramenter,*matrixB)
Note:
                A + &B   <===>  A .operator+ (&B)
                  ^^^               ^^^^^^^^^^^^^^^^^^^^
 
Back to your problem.
You decalare two Matrix objects.
            matrix1<int> *A=new matrix1<int>(row,col);
            matrix1<int> *B=new matrix1<int>(row,col);
 
and use following instruct to do + operation.
            matrix1<int> *C=A + B; // <<---   Error
There is a error on the instruction!
 
Did you remember the note:
      A + B <===>  A.operator+(B)
But you defined the variable A as a pointer type of matrix1<int>, that is
          matrix1<int> *A=new matrix1<int>(row,col);
You cannot use "." to reference a class member function but use "->" to reference it.
 
That is why the "matrix1<int> *C=A->operator+(B)" is work and "matrix1<int> *C=A+B " don't.
 
So the solution is that
           matrix1<int> *C=(*A)+B;
