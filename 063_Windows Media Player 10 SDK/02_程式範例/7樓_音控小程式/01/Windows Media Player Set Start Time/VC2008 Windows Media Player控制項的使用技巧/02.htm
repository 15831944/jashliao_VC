VC2008 Windows Media Player控件的使用技巧(二)

分类： 2.1 VC++/MFC 2009-11-29 15:07 8314人阅读 评论(58) 收藏 举报

操作系统：Windows 7
软件环境：Visual C++ 2008 SP1
本次目的：在VC2008下Windows Media Player控件的用法之实用篇

      很奇怪，也许是WMP.DLL文件的问题，虽然Windows Media Player正常，但是在VC2008中插入Windows Media Player控件后，却在要为控件添加类的时候，找不到这个控件的类，如下所示：

本文将介绍一种通用的方法，完成Windows Media Player控件的实现。首先，在这里 下载Windows Media Player控件的所有类(24个)，有看上一篇 的话，就可以知道我为什么那时候生成了全部的类了。现在把那些类添加到工程，当然依旧根据需要添加，不过CWMPPlayer4.h和CWMPPlayer4.cpp最好添加进去，因为那是跟音视频连接的。点击“项目 ”→“添加现有项 ”，选择需要的类，“添加 ”完成。

接下来就是添加头文件、声明控件对象和绑定控件了，
在TestDlg.h 头文件添加    #include "CWMPPlayer4.h" ，
然后声明      CWMPPlayer4 m_Player;
接着在TestDlg.cpp : 实现文件中

  1. void CTestDlg::DoDataExchange(CDataExchange* pDX)  
  2. {  
  3.     CDialog::DoDataExchange(pDX);  
  4.     DDX_Control(pDX, IDC_BUTTON1, m_Open);  
  5.     DDX_Control(pDX, IDC_OCX1, m_Player);//绑定播放器控件  
  6. }  

最后，我们可以测试了，在这个工程里我加了一个按钮，实现打开播放的。

  1. void CTestDlg::OnBnClickedButton1()  
  2. {  
  3.     // TODO: 在此添加控件通知处理程序代码  
  4.     m_Player.put_URL(_T("E://Music//铃声//说好的幸福呢(铃声).mp3"));  
  5. }  

效果已经出来了，至于剩下的对播放器的控制，如：打开、播放、暂停、快进等等，下一篇将会介绍。

windows Media Player控件类打包： http://download.csdn.net/source/1856942
