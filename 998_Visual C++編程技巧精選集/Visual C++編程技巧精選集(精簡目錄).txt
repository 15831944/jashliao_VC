內容簡介：

    本書及其配套光盤使用完整的源代碼列舉了1358條實用性極強的Visual C++ (標準C++)編程技巧，旨在幫助廣大讀者快速解決實際開發過程中面臨的諸多問題，從而提高程序開發效率、拓展軟件應用領域。全書內容包括：消息框和對話框、常用控件和組件、標題欄和菜單欄、工具欄和狀態欄、光標和圖標應用、程序主窗口界面、程序管理和控制、打印機和打印實現、進程和線程管理、文件讀寫操作、文件和文件夾管理、字符串和數據轉換、操作系統維護、計算機設備管理、文字和圖像顯示、音頻和視頻、網絡通信、數據庫管理、數據庫訪問、數據庫查詢、Word和Excel應用、WMI查詢和管理、注冊表綜合應用等多個部分。為了突出編程技巧的實用性和簡潔性，本書在演示或描述編程技巧時，力求針對性地解決問題，盡量減少和避免問題的復雜化。由于篇幅限制，本書文字部分中僅給出與此編程技巧密切相關的主要代碼，完整的源代碼請參考配套光盤。
　　本書適于作為Microsoft Visual Studio 2005(Visual C++ 2005)用戶的案頭參考書，無論是編程初學者，還是編程高手，本書都極具參考和收藏價值。
